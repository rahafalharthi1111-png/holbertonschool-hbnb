## Users API
- POST /users → 201 Created (valid data)
- POST /users → 400 Bad Request (missing email)
- GET /users → 200 OK

## Places API
- POST /places → 201 Created
- POST /places → 400 Bad Request (missing name)

## Reviews API
- POST /reviews → 201 Created
- POST /reviews → 400 Bad Request (missing text)

## Amenities API
- POST /amenities → 201 Created
- POST /amenities → 400 Bad Request (missing name)
